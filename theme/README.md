# Sierra Style - California 89 Shopify Theme

A modern, rustic Shopify theme inspired by mountain living and the California 89 lifestyle.

## Theme Structure

```
theme/
├── assets/
│   ├── css/
│   │   ├── base.css          # Base styles and reset
│   │   ├── theme.css         # Main theme styles
│   │   ├── components.css    # Component styles
│   │   ├── utilities.css     # Utility classes
│   │   └── responsive.css    # Responsive styles
│   ├── js/
│   │   ├── global.js        # Global utilities
│   │   ├── theme.js         # Main theme JS
│   │   ├── cart.js          # Cart functionality
│   │   ├── slider.js        # Hero slider
│   │   ├── filters.js       # Collection filters
│   │   ├── product.js       # Product interactions
│   │   ├── menu.js          # Mobile menu
│   │   └── animations.js    # Scroll animations
│   └── images/              # Image assets
├── config/
│   └── settings_schema.json # Theme settings
├── layout/
│   └── theme.liquid         # Main layout
├── sections/                # Theme sections
├── snippets/                # Reusable snippets
└── templates/               # Page templates
```

## Features

- **Responsive Design**: Mobile-first approach with breakpoints
- **Hero Slider**: Auto-playing carousel with touch/swipe support
- **AJAX Cart**: Slide-out cart drawer with real-time updates
- **Product Filters**: Collection page filtering and sorting
- **Mobile Menu**: Slide-out navigation menu
- **Product Interactions**: Quick view, wishlist, add to cart
- **Scroll Animations**: Fade-in animations on scroll
- **Customizable Sections**: All sections editable in theme editor

## Installation

1. Upload the theme folder to your Shopify store
2. Go to Online Store > Themes
3. Click "Add theme" > "Upload zip file"
4. Customize the theme in the theme editor

## Customization

### Theme Settings

Navigate to Theme Settings to customize:
- Colors
- Typography
- Header settings
- Footer settings

### Sections

All sections are customizable in the theme editor:
- Hero Slider
- Feature Icons
- Featured Products
- Shop by Category
- Customer Testimonials
- Blog Posts
- Instagram Feed

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Development

To develop locally:
1. Install Shopify CLI
2. Run `shopify theme dev`
3. Make changes and see live updates
